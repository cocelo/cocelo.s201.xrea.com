* screen [#o58120df]
screen は、例えば SSH などによる遠隔操作で何かのソフトを make 中に不意に切断されても処理は続けられて、かつ再接続出来る、というものです。

* インストール [#k74ec406]
上記でインストールした sudo を早速使ってみましょう。

 % sudo portinstall sysutils/screen
 % rehash
